# AI News Aggregator with Summarization

A complete full-stack web application that aggregates AI news from top tech websites and provides concise summaries using AI. Built with the MEAN/MERN stack (MongoDB, Express, Angular/React, Node.js) and Python.

## Architecture

The project is built using a modern microservice-like architecture separating frontend, backend, and the AI processing service.

1. **Frontend**: The user interface is developed for a responsive, modern experience.
2. **Backend (Node.js & Express)**: A RESTful API built with **Express** and **Mongoose** to handle user authentication (JWT), data persistence in MongoDB, and route requests.
3. **AI Service (Python)**: A specialized service leveraging LLMs to provide advanced text summarization features.

## Getting Started

1. Set up the Python scraping service.
2. Configure environment variables for the MERN Stack application.
3. Start the node backend and frontend development servers using the `start.bat` script.
